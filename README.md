# Forza Horizon 5 Saveswapper

A modern tool for swapping XUID in Forza Horizon 5 save files.

![Forza Horizon 5 Saveswapper](https://img.shields.io/badge/Game-Forza%20Horizon%205-blue)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

## Features
- **Easy to Use** - Simple drag & drop or file picker

## Supported 

- Forza Horizon 5 

## How to Use

1. **Download** the latest release from [Releases](../../releases)
2. **Run** `Saveswapper.exe`
3. **Select** your save file (`.ProfileData`)
4. **Enter** your new XUID
5. **Click** "Export Modified Save"
6. **Done!** Your modified save is ready

## Finding Your Save Files

Forza Horizon 5 save files are typically located at:
```
C:\Users\[YourUsername]\AppData\Local\Packages\Microsoft.SunriseBaseGame_[ID]\SystemAppData\wgs\[FOLDER]\[FOLDER]\
```

Look for files named `User_[XUID].ProfileData`

## Finding Your XUID

1. Press `Win + R`
2. Type `ms-settings:privacy-accountinfo` and press Enter
3. Your XUID is displayed in the Account Info settings

Or use online tools to convert your Gamertag to XUID.


## Important Notes

- Always backup your original save files before modifying
- This tool only works with save files from supported game versions
- Modified saves are for personal use only
